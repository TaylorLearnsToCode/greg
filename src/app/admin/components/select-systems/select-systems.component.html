Click "Load" to replace the active data with the target system.
Click "Augment" to add the selected system to the communal pot: such that the configurations for both coexist in memory.

Systems highlighted in grey are not fully functional and may not be implemented - but are on the horizon.

Current state of the program provides for generation of dungeons - top to bottom - using the LBB ruleset. Some limited capacity is also feasible, selecting the different mechanisms for generation on the dungeon panel (e.g. - you can use the B/X generation procedures with the LBB monster and treasure tables if you like).

The next milestone for data is going to be the completion and inclusion of B/X configuration data - however, as parallel developments on WW&W as a system occur, the WW&W features and data will likely perforate the B/X development schedule.
